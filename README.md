tmux based workflow

```sh
tmux
# Ctrl-b % to split into two panes
# Ctrl-b o to switch to original pane

# run the server
./gradlew clean run

# Ctrl-b o to switch to the second pane
./gradlew HelloClient

# Ctrl-b o to switch to original pane
# Ctrl-c to kill the server

# Ctrl-b x to kill panes and tmux session
```
